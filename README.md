# OASIZ_TimeLogger2

時間記録支援アプリケーション。
([mikoto2000/OASIZ_TimeLogger: 時間記録支援アプリケーション](https://github.com/mikoto2000/OASIZ_TimeLogger/tree/master) の再実装)


# Features:

- [x] : 作業名を指定して、作業時間を記録すること
- [ ] : 記録した作業を後から確認すること
- [x] : 作業名を後から変更すること
- [ ] : 作業記録をエクスポートすること
